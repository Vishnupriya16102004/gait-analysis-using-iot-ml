# gait-analysis-using-iot-ml
🧠 Gait Analysis using IoHT and Machine Learning
📌 Overview

This project focuses on analyzing human gait patterns using IoHT-based sensor data. The system uses IMU and EMG signals to study movement and detect irregular patterns.

⚙️ Technologies Used
Python
Machine Learning (Isolation Forest)
IMU & EMG Sensors
Data Processing
📊 Features
Processes real-time sensor data
Handles noisy and inconsistent signals
Detects abnormal gait patterns
Supports basic health monitoring analysis
🧪 Dataset
Includes sample sensor data (IMU & EMG signals)
Preprocessed for machine learning analysis
🚀 How to Run
Load the dataset
Run the Python script
View output and anomaly detection results
📈 Outcome

⚠️ Challenges Faced
Noisy and inconsistent sensor data
Difficulty in extracting stable patterns from real-world signals

The project demonstrates how sensor data can be analyzed using machine learning to detect irregular gait patterns, highlighting challenges in real-world data reliability.
